# tourist-attractions-App
    地元の観光名所を紹介するWebアプリです。  
    Flask を用いて作成し、観光地の登録・表示、画像管理を行います。

## フォルダ構成

- tourist-attractions-App/
- - .git/                   :Gitのバージョン管理用フォルダ
- - docs/                   :設計書や仕様書などの開発資料を保存するフォルダ
- - static/                 :CSSや画像などの静的ファイルを保存するフォルダ
- -   css/                  :スタイルシートを保存するフォルダ
- -   img/                  :サイトで使用する画像を保存するフォルダ
- -     common/           :ロゴや背景画像など、共通で使用する画像を保存するフォルダ
- -     uploads/          :観光地画像を保存するフォルダ
- - templates/              :Flaskで使用するHTMLテンプレートを保存するフォルダ
