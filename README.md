# tourist-attractions-App
    地元の観光名所を紹介するWebアプリです。  
    Flask を用いて作成し、観光地の登録・表示、画像管理を行います。

## フォルダ構成と役割

| フォルダ名 | 役割 |
|---|---|
| `.git/` | Git のバージョン管理情報を保存するフォルダ |
| `docs/` | 設計書や仕様書などの開発資料を保存するフォルダ |
| `static/` | CSS や画像などの静的ファイルを保存するフォルダ |
| `static/css/` | スタイルシートを保存するフォルダ |
| `static/img/` | サイト内で使用する画像を保存するフォルダ |
| `static/img/common/` | ロゴや背景画像など共通画像を保存するフォルダ |
| `static/img/uploads/` | 観光地画像を保存するフォルダ |
| `templates/` | Flask で使用する HTML テンプレートを保存するフォルダ |
